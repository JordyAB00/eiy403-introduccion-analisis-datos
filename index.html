Información del Curso

Objetivo: Proporcionar a estudiantes de diferentes áreas disciplinarias los conocimientos necesarios para introducirse al análisis de datos usando un lenguaje de programación con utilidad estadística.


Metodología: Aprendizaje activo con enfoque práctico e interdisciplinario.


Evaluación:

  • Laboratorios: 40%
  • Investigación: 25%
  • Tareas: 25%
  • Presentación: 10%

Recursos Adicionales

Repositorio GitHub: Todos los materiales del curso están disponibles en nuestro repositorio oficial.


Estructura:

  • Presentaciones semanales
  • Notebooks de laboratorio
  • Datasets para práctica
  • Tareas y proyectos
  • Referencias bibliográficas

Historia del análisis de datos, lenguajes de programación (Python/R), análisis exploratorio básico y medidas estadísticas fundamentales.

Clase 2 (6 de Agosto)
Análisis Exploratorio de Datos (EDA)

Clase 3 (13 de Agosto)

Estructuras de datos aplicadas a análisis químicos, importación de datos desde instrumentos analíticos, primer dataset de control de calidad químico real.
